# Events-App-V3
Sprint 2 of our Events App

## Usage
```
$ git clone <repository>
$ cd Event-App-V3
$ cd server
$ npm install
$ npm start
$ cd ..
$ cd client
$ npm install
$ npm start
```

You may have an issue with linking to MongoDB because your IP address isn't listed in the allowed addresses.
